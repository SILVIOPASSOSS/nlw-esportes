## Start

**Server**

Primeiramente,
Crie um novo arquivo `.env ` usando o `.env.example`
Então, crie o banco de dados local:

```
$ npm install
$ npm run db:migrate
```

Depois disso, basta startar o server:

```
$ npm run dev
```

**Web**

Para abrir o Cliente Vite localmente

```
$ npm run install
```
